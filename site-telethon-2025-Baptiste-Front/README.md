# Telethon 2025 Front-end

Il s'agit d'un projet Nuxt4 avec les modules :

- nuxt-auth-utils pour le routage

Pour démarer le projet il faut avoir Node et npm puis faire npm install à la racine.

## Liste des commandes :

- npm run dev pour lancer le projet en mode développement
- npm run build pour construire le projet
- node ./output/index.js pour lancer le serveur une fois construit

## Admin
